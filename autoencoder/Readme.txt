	
1.Project 
 - Reproducing Autoencoder on MNIST 

2.Description
 - Using the batch normalization, Autoencoder is implemented on MNIST, showing the comparison of original image and output image through the auto-encoder.

3.Date & Author
 - Aug 2017, Gonsoo Moon

4.Environment
 1) Mac Pro 2.6 GHz Intel Core i5, 16 GB RAM
 2) Tensorflow 1.0.0
 3) Python 3.6

5.How to run
 On the command line, type the following
 python autoencoder_mnist.py 4
 Note:
 # number of neurons in the code layer
 On another command line window, type, for tensorboard,
 tensorboard --logdir = ./mnist_autoencoder_hidden=4_logs
