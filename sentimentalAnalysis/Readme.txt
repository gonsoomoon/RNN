	
1.Project 
 - Reproducing sentimental analysis on IMDB

2.Description
 - Using Dropout LSTM (Long Short Term Memory), sentimental prediction on the IMDB movie reviews is implemented

3.Date & Author
 - Sep 2017, Gonsoo Moon

4.Environment
 1) Mac Pro 2.6 GHz Intel Core i5, 16 GB RAM
 2) Tensorflow 1.0.0
 3) Python 3.6

5.How to run
 1) Download imdb from http://www.iro.umontreal.ca/~lisa/deep/data/imdb.pkl
 2) Make data folder and copy the imdb into it.
 2) Install tflearn using pip install tflearn
 3) python imdb_lstm.py
